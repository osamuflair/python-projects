# Authentication API

## What is This Project
- It is a FastAPI authentication system that demonstrates JWT token-based authentication and protected routes

## Technologies
- Python
- FastAPI
- PyJWT
- pwdlib
- Thunder Client

## Features
- Password hashing
- User verification
- Login endpoint that returns a JWT token
- Protected route that requires a valid token

## How to run
- Install dependencies
- Run main.py using:
    - uvicorn main:app --reload
- Test end points using Thunder Client
