# Contact API

## What is This Project
It is a project that demonstrates how the different end points - GET, POST, PUT, and DELETE works.
A list of dictionaries is used as the database

## Technologies
- Python
- flask
- Thunder Client

## Features
- GET: Gets data from database
- POST: Adds data to the database
- PUT: Updates data on the database
- DELETE: Removes data from the database

## How to run
- Install flask
- Install Thunder Client
- Run app.py
- Test end points individually using Thunder Client
