# Inventory Management System

## What is This Project
It is a project that helps businesses keep track of their stocks

## Technologies
- Python
- PostgreSQL
- psycopg2

## Features
- Add different products
- View all products
- Update specific product
- Delete specific product
- Search for specific product
- Gives alerts for low stock

## How to run
- Install psycopg2
- Set up PostgreSql
- Create a database
- Run local.session.sql to create the tables and insert data
- Run inventory_management.py to start the program
