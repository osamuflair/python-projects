# Contact Manager

## What is This Project
- It is a project that demonstrates how the different end points - GET, POST, PUT, and DELETE works using fastAPI.
- It shows how easier it is to use FastApi
- A list of dictionaries is used as the database

## Technologies
- Python
- FastAPI
- Thunder Client

## Features
- GET:
    - Gets all data from database
    - Gets data from specific id
    - Search for data by name from database
- POST: Adds data to the database
- PUT: Updates data on the database
- DELETE: Removes data from the database

## How to run
- Install FastAPI
- Install Thunder Client
- Run main.py using:
    - uvicorn main:app --reload
- Test end points individually using Thunder Client
